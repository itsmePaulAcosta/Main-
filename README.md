# Portfolio Website

A modern, responsive portfolio website inspired by creative web design trends. This portfolio showcases skills, services, testimonials, and provides a contact form for potential clients.

## Features

- **Responsive Design**: Works perfectly on all devices (mobile, tablet, desktop)
- **Modern UI/UX**: Clean, dark theme with green accent colors
- **Smooth Animations**: CSS animations and JavaScript interactions
- **Interactive Elements**: 
  - Animated counters
  - Smooth scrolling navigation
  - Mobile hamburger menu
  - Contact form with validation
  - Scroll-to-top button
- **Sections Included**:
  - Hero section with animated typing effect
  - Statistics counters
  - Services showcase
  - About section with tabs
  - Skills and tools display
  - Testimonials
  - Contact form
  - Social media links

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript (ES6+)**: Interactive functionality
- **Font Awesome**: Icons
- **Google Fonts**: Poppins font family

## Customization Guide

### 1. Personal Information
Update the following in `index.html`:
- Replace "YOUR NAME" with your actual name
- Update the hero description
- Change contact email and location
- Update social media links

### 2. Content Sections
- **Services**: Modify the services you offer
- **About**: Update your personal story and experience
- **Skills**: Add your specific skills and tools
- **Testimonials**: Replace with your actual client testimonials
- **Statistics**: Update the counter numbers to reflect your experience

### 3. Styling
- **Colors**: Main accent color is `#00ff88` (green). Change this in `styles.css`
- **Fonts**: Currently using Poppins. Change the Google Fonts link in HTML
- **Images**: Add your photos to the `images/` folder and update references

### 4. Images Needed
Place these images in the `images/` folder:
- `profile.jpg` - Your profile photo
- `resume.pdf` - Your resume file
- Any project screenshots or portfolio images

## Setup Instructions

1. **Download/Clone** the files to your local machine
2. **Customize** the content with your personal information
3. **Add Images** to the images folder
4. **Test Locally** by opening `index.html` in a web browser
5. **Deploy** to your preferred hosting platform (GitHub Pages, Netlify, Vercel, etc.)
